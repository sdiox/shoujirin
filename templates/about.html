About and why you will love this

love the dictionary database; hate the website

@sdiox (the guy who created this web app)

If you are Thai and need to use your Japanese language skill for some reason, having a nice Thai <> Japanese dictionary would be very handy. Ain't it?

And J-doradic is the online dictionary website I frequently use because it has, probably, the biggest database of Japanese words.

The only problem with the website is that the website layout contains so many unnecessary elements which have nothing to do with the purpose of my use of this website. Therefore, I decided to create a website that will forward a lookup request to J-doradic, parse the result, and present it to users in a clean-looking manner.



This web app is developed using Flask framework and BeautifulSoup4.
